* les répertoires n40, n50, n100 contiennent les instances du problème. Pour chaque n, 125 instances sont données et numérotées de 1 à 125
	- Exemple : le fichier n100/n100_1.txt est le fichier contenant les données de l'insistance numero 1 pour n=100

* Chaque instance contient n lignes, chaque ligne i contient trois valeurs correspondant respectivement à pi (le temps d'execution), wi (le poids) et di (la date limite) 

* les valeur optimales sont données dans les fichiers wt40.txt, wt50.txt, et wt100.txt. Chaque ligne i correspond à la valeur optimale de l'instance numero i.
	- Exemple : la première ligne du fichier wt100.txt contient la valeur de la solution optimale de l'instance donnée dans n100/n100_1.txt

* Pour les questions 'challenge' pour n=1000, le fichier des données est le même que précédemment.
